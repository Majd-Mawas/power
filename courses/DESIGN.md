# Design System Documentation

This document outlines the core aesthetic and functional principles of our design system.

## Theme Overview

Our design theme is built to be **light** and vibrant.

## Color Palette

We utilize a **Fidelity** color variant, providing a distinct and modern feel.

- **Primary Color**: `#06B6D4` - A vibrant blue, used for primary actions and key interactive elements.
- **Secondary Color**: `#0a1e3b` - A deep, dark blue, providing strong contrast and depth.
- **Tertiary Color**: `#006c47` - A rich green, used for accents, highlights, and decorative elements.
- **Neutral Color**: `#0F172A` - A very dark blue, serving as a base for backgrounds and text.

## Typography

Our typography system ensures clarity and readability across all content.

- **Headlines**: `plusJakartaSans` - Provides a modern and energetic feel for headings.
- **Body Text**: `inter` - A highly readable and versatile font for all body content.
- **Labels**: `inter` - Consistent with body text for clear and concise labels.

## Shape and Spacing

- **Roundedness**: `2` (Moderate) - UI elements feature a balanced level of roundedness, offering a friendly yet professional aesthetic.
- **Spacing**: `2` (Normal) - We maintain a normal level of whitespace, ensuring content is well-organized without feeling cramped or overly sparse.
